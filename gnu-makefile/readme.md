### 1. Supported Makefile Build Rules

- **Clean the project and delete the build directory**:
  ```bash
  make clean
  ```
- **Build the Project**:
  ```bash
  make all
  ```
- **Parallel Build with -jN: N is number of threads to run**:
  ```bash
  make all -j4
  ```
