### 1. Supported Makefile Build Rules

- **Clean the project and delete the build directory**:
  ```bash
  make clean
  ```
- **Build the Project**:
  ```bash
  make all
  ```
- **Paralle Build with -jN: N is number of thread to run**:
  ```bash
  make all -j4
  ```
  
### 2. Tools Path to Update
Update the tools path `tools` in `tools.mk`
